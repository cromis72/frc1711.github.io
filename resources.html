Installations

  • Eclipse

    • Eclipse is our main IDE, and we will use it for all in-season programming work. Eclipse is free, and required for programming work on this team.

  • Eclipse for FRC Setup

    • These are FRC specific setup instructions for Eclipse, including additional downloads and configurations

  • Git

    • Git is used to interact with our repositories. You can use Git through the command line.

  • Ruby Installfest

    • Ruby is another language that we may use in the offseason. This link leads you through the steps necessary to get started with Ruby, and Ruby on Rails. While this is completely optional, it is recommended if you want to participate in a broader range of offseason projects.

Tutorials

  • Git Setup

    • This article on Guthub will walk you through the steps of setting up Git for your command line

  • FRC Setup

    • This is a reference containing all the information about the FRC control system setup, including FRC-specific hardware and drivers. You most likely will not need to install any of this on your personal laptop.

  • Codecademy

    • Codecademy has tutorials for several different languages. The Java and Command Line tutorials are required for this team, and there are other useful tutorials if you are interested.
